GuardedAsyncTask

abstract class GuardedAsyncTask<Params, Progress> : AsyncTask<Params, Progress, Void>

Abstract base for a AsyncTask that should have any RuntimeExceptions it throws handled by the JSExceptionHandler registered if the app is in dev mode.

This class doesn't allow doInBackground to return a results. If you need this use GuardedResultAsyncTask instead.
