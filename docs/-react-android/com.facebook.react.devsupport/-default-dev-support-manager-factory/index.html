DefaultDevSupportManagerFactory

class DefaultDevSupportManagerFactory : DevSupportManagerFactory

A simple factory that creates instances of DevSupportManager implementations. Uses reflection to create BridgeDevSupportManager if it exists. This allows ProGuard to strip that class and its dependencies in release builds. If the class isn't found, PerftestDevSupportManager is returned instead.
