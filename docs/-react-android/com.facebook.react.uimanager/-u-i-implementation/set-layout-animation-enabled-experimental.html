setLayoutAnimationEnabledExperimental

open fun setLayoutAnimationEnabledExperimental(enabled: Boolean)

LayoutAnimation API on Android is currently experimental. Therefore, it needs to be enabled explicitly in order to avoid regression in existing application written for iOS using this API.

Warning : This method will be removed in future version of React Native, and layout animation will be enabled by default, so always check for its existence before invoking it.
