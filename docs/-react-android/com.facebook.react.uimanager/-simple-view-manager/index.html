SimpleViewManager

abstract class SimpleViewManager<T : View> : BaseViewManager<T, LayoutShadowNode>

Common base class for most of the {@link ViewManager}s. It provides support for most common properties through extending {@link BaseViewManager}. It also reduces boilerplate by specifying the type of shadow node to be {@link ReactShadowNode} and providing default, empty implementation for some of the methods of {@link ViewManager} interface.
