@JvmStatic
fun sendTouchEvent(eventEmitter: RCTModernEventEmitter, event: TouchEvent)

Generate touch event data to match JS expectations. Combines logic in sendTouchEvent and FabricEventEmitter to create the same data structure in a more efficient manner.

Touches have to be dispatched as separate events for each changed pointer to make JS process them correctly. To avoid allocations, we preprocess touch events in Java world and then convert them to native before dispatch.
